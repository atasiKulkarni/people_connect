"People Connect" is a comprehensive, full-stack Human Resource Management System (HRMS) designed to centralize employee engagement and administrative functions through web and mobile applications. Built with React.js, Node.js, and PostgreSQL, the platform streamlines employee lifecycle processes from attendance to compensation.
Key features include:
-SSO Login & Directory: Secure Single Sign-On and an interactive employee directory for easy navigation and connection.
-Time & Attendance Management: Employees clock in/out daily, fill timesheets, apply for leave, and view pending requests and work hour averages.
-Employee Engagement ("Engage" Page): A dedicated social feed where employees can post, like, and comment, alongside official HR/Admin announcements.
-HR Portal: A central hub for HR policies, document management (offer letters, revised letters), and company vacancy postings.
-Compensation & Rewards: Modules for viewing payslips, tax information, and tracking employee rewards and recognition.
-Integrated Tools: Features a chatbot and a dynamic banner for company-wide communication.
-Unified Admin Panel: A single, centralized HR administration panel manages all content and data for both the web application and the React Native mobile app.
